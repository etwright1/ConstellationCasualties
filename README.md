# ConstellationCasualties

This repository contains the data, codes and plots used in the paper 'Satellite megaconstellations and collective casualty risks' (2026) by Ewan Wright, Aaron Boley and Michael Byers, published in Science Policy. 

It is provided for independent assessment. If you use information provided here, please cite the 2026 paper linked here: [tbc].

Figure 1 was generated in Microsoft Excel in the file 'Casualty risk thresholds plot data.xlsx'.

The constellation causalty risk was calculated using the script 'main file.py' which uses 'KeplerTools.py' and 'casualty.py' from our previous work.
